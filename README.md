This project is for check Authentication security

## To run project

-  Go to back_end and open cmd and run

```bash
db.sh
```

to create database and then run nodejs :

```bash
npm i
npm start
```

and then run frontend (in this case reactjs):

```bash
cd front_end
npm i
npm start
```
